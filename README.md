# RunUntil Technologies GmbH

Corporate website for RunUntil Technologies GmbH.

## Setup

This site is hosted on GitHub Pages.

### Custom Domain

To set up a custom domain:
1. Add your domain to `CNAME` file
2. Configure DNS with your domain provider:
   - For apex domain: Add `A` records pointing to GitHub Pages IPs
   - For www subdomain: Add `CNAME` record pointing to `rununtil.github.io`
